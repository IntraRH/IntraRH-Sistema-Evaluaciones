🗄️ Base de datos Supabase
✅ Conectado. Los datos se sincronizan automáticamente.
⚠️ Importante: Para proteger los datos, verifica que todas las tablas tengan RLS activado en tu dashboard de Supabase → Authentication → Policies. Sin RLS los datos son accesibles públicamente.
🛡️ Respaldo local de respuestas
Cada respuesta completada genera automáticamente una copia local por proyecto (irh_backup_{proyectoId} en localStorage). Si Supabase falla, este respaldo permite recuperar los datos.
